My Work

Alarm System

The project that motivated me to start coding. This is the frontend part of my home's alarm system. My RasperryPi is sending sensor codes to a database whenever doors in my home open or close. In the frontend part of the system all alarms are displayed. It also allows me to create more alarm systems and add sensors to them. Of course the private alarm system data is hidden behind a login. With this, other users could also log in and manage their own alarm systems.
